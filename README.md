# Mega Supermarket Sales Analytics Project

## Overview
This project focuses on analyzing Mega Supermarket sales data using Python and Power BI. The project includes data cleaning, visualization, dimensional modeling, and dashboard development for business intelligence.

---

## Technologies Used
- Python
- pandas
- matplotlib
- Power BI
- Excel

---

## 📊 Features
- Sales analysis
- Customer behavior analysis
- Payment method analysis
- Gross income analysis

## 📈 Visualizations
The project includes:
- Bar charts
- Pie charts
- Line charts
- Histograms

## Key Insights
- Identified the highest-performing branch
- Analyzed customer payment preferences
- Determined the most profitable product lines
- Tracked sales trends over time
